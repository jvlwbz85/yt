## 产品线拆解

云途目前主要有以下几条产品线：

### 1. 香港三网优化云主机（HK.AMD系列）

这是最热门、也最容易被抢空的一条线。

三网优化，意思是电信、联通、移动三家运营商都有针对性路由优化，不是那种随缘绕路的国际线路。经过实测：
- **电信**：去程CN2，回程走联通CUG（10099高速直连）
- **联通**：去程4837，回程CUG（10099）
- **移动**：去程回程均走CMI直连

晚高峰实测带宽可以跑到150Mbps以上，博客类网站平均加载时间可压缩到0.5秒左右，体验相当不错。

有一点需要说明：**优化云不承诺100%三网直连路由**，极端负载下可能临时切到其他大陆优化路由。如果需要承诺路由稳定，建议选择**香港精品云主机**系列（价格略高一档）。

### 2. 香港BGP国际线路 + 送CNIX入口

这是最近一段时间圈子里讨论度最高的产品。

采用AMD EPYC 7542平台 + NVMe SSD阵列，自带香港原生IPv4，最关键是**免费赠送50个CNIX专线入口**。

CNIX是深圳前海的数据交换中心，专为内地和香港之间的高速互联设计，走内网通道而非公网绕路。有了CNIX入口，就算机器本身是国际线路，连接质量也能大幅提升。

花"国际线路VPS"的钱，享受"接近专线"的连接稳定性，性价比这一块确实有点绝。

### 3. 深圳电信云主机

高频CPU平台，直连深圳前海CNIX，适合需要深港两地低延迟互通的场景。

### 4. CNIX深港NAT专线

纯专线加速产品，A端深圳、Z端香港，默认20个端口，适合需要大带宽深港互联、自备大厂前置机的高级用户。

### 5. 香港独立服务器

目前有HK.DS.F方案，Intel Xeon E5-2630 V2处理器、32GB内存、480GB SSD，可选1Gbps国际线路限流10TB或500Mbps优化线路，350元/月起步。

## 几点注意事项

1. **优化云 ≠ 精品云**：优化云只承诺尽量优化，极端负载下可能会临时换路由；精品云才明确承诺三网直连路由，两者价格有差异。
2. **国际BGP线路 ≠ CN2优化**：HK.BGP系列是纯国际线路，回内地可能走CMI，不保障大陆优化效果，更适合外贸、跨境或海外访问场景。
3. **CNIX产品需要自备前置机**：深港CNIX专线和NAT系列产品，需要额外搭配一台国内大厂的云服务器做入口前置，直接购买后无法单独使用。
4. **商家入场较新**：2025年成立，体量不如老牌服务商，选择月付或短期套餐可以降低风险。
